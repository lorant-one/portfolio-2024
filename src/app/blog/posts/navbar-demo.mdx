## Navbar demo

Once UI [handles themes](https://once-ui.com/docs/theming) on a global level as a set of custom data attributes. They control color profiles, roundess, solid style, and more. Pasting a component built with the standard Once UI tokens will inherit all style properties of your project's settings.

Here's a navigation that I built a while ago:

Let's modify some attributes to style it in a slightly different way while keeping the main characteristics of the component:

<CodeBlock
    maxWidth="s"
    codePreview={
        <Flex fillWidth
            data-theme="light"
            data-neutral="slate"
            data-border="rounded"
            data-solid="color"
            data-solid-style="plastic">
            <Navbar/>
        </Flex>
    }
    codeInstances={[
        {
            language: 'tsx',
            label: 'Next.js',
            code:
`<span
    data-neutral="slate"
    data-border="rounded"
    data-solid="color"
    data-solid-style="plastic">
    <Navbar/>
</span>`
        }
    ]}/>

<br/>
It's important to mention that data attributes can be nested as well. If you want to style certain elements in a different way, you can simply define different data attributes on the element itself.
